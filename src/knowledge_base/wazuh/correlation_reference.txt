OVERVIEW
--------------------------------------------------------------------------------
Correlation combines multiple related events into a higher-confidence
security detection.

Correlation Dimensions:

Time

Source

Destination

User

Host

================================================================================
TEMPORAL CORRELATION
================================================================================

frequency

Number of required events.

Example:

<frequency>10</frequency>

--------------------------------------------------------------------------------

timeframe

Observation window.

Example:

<timeframe>300</timeframe>

Meaning:

10 events within 300 seconds.
